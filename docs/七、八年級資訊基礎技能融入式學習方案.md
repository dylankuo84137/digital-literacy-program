# **七、八年級資訊基礎能力與素養融入式學習方案 (V2)**

**【版本紀錄】**

* **V1 (20251020)：** 著重於「工具融入式」學習，強調透過 Google Docs 等雲端工具完成任務（例如：「班級詩集」排版）。  
* **V2 (會議修訂版20251029)：** 依據資訊領域會議決議，方案核心**轉向「素養優先、工具降載」**。

### **一、 問題背景與核心目標**

* **問題背景：**  
  1. **技能落差：** 經觀察，九年級學生在資訊應用能力上，特別是中英文打字速度與正確率，存在極大的個別差異。  
  2. **素養與技術不同步：** 教師擔憂「技術導入過早、素養未同步」，學生雖能操作 3C，卻缺乏應有的數位公民意識與媒體思辨能力。  
  3. **學習能力衝擊：** 憂慮「數位化學習可能削弱語文能力」，且學生在家庭中普遍「缺乏引導與節制」。  
* **核心理念：引導探索歷程，建立數位心態** 本方案旨在回應上述挑戰，建立一套概念性的學習路徑。我們相信，資訊教育的核心價值在於「**探尋與修正」的歷程，而非僅是「精準搜尋」的結果**。此方案可視為一個「儀式性意義」，是帶領學生開啟面對資訊世界學習之門的重要起步。  
* 總體目標：  
  面對升上國中、生活與學習更加自主的階段，本方案的總體目標，不僅是讓學生在九年級前具備打字等基礎技能，更重要的是，引導他們建立健康的數位使用心態。我們期望學生從「被動的資訊消費者」轉變為「主動的知識創造者」，學習駕馭數位工具以完成有意義的任務、有效整理資訊、並進行團隊協作，正面回應 3C 議題所帶來的挑戰與機遇。

### **二、 實施策略：跨年級、跨領域的融入式學習**

為確保方案能全面實施，達成全校學生基本能力養成的目標，共識是此方案需從「**領域**」的角度推動，以確保推動的廣度與一致性。

* **執行模式：** 由**資訊老師為主導，七、八年級導師與各領域科任老師為協作者**，分年級、分階段逐步實施。  
* **實施時間：** 為求集中效益並減少干擾，主要利用**七、八年級在四季其中有多堂導師課的季，集中 4\~5 節課**的時間，讓資訊老師入班協同教學。

### **【七年級：基礎素養與線下能力扎根階段】**

### **第一階段：「數位公民」素養導入（工具使用前）**

* **目標：** 在開通帳號、接觸工具前，優先建立「素養先於技術」的共識。  
* **建議內容：**  
  1. **認識網路世界：** 介紹「數位足跡」與「數位公民」的概念，理解其意涵。  
  2. **覺察網路言行：** 透過 CS Unplugged 活動或案例討論，模擬網路世界的互動，覺察個人在網路上的行為與形象，並對其產生的影響有所感受。

### **第二階段：「數位身份」建立與基礎工具探索**

* **目標：** 建立基本帳號認知，並練習有意義的基礎操作。  
* **建議內容：**  
  1. **我的數位身份證 (smail)：** 在具備數位公民概念後，才帶領全班登入 smail 帳號。再次強調「網路世界的身份」概念、帳號密碼保管的重要性，以及網路言行需等同現實生活中的行為。  
  2. **鍵盤的探索：** 保留熟悉打字能力。介紹正確坐姿，利用線上打字遊戲（如 typing.com）讓學生在遊戲中熟悉鍵盤。

### **第三階段：「探尋歷程」主題式任務 (強化線下)**

* **目標：** 將技能學習拆解成有溫度的任務，並強化「手寫」與「紙本」的整理能力，回應「數位學習削弱語文能力」的擔憂。  
* **任務範例：「圖書館尋寶」 (手寫版)**  
  * **情境：** 從實用需求切入，引導學生使用圖書館的圖書搜尋系統。  
  * **任務：** 找到書籍後，**必須將查找的書籍資料（如：書名、作者、索書號、內容摘要）以「手寫」方式整理與紀錄** 在工作本上，練習資訊的「輸入」與「輸出」能力。

### **第四階段：「數位技能護照」**

* **目標：** 透過遊戲化方式讓學生看見自己的成長。此活動為鼓勵性質。  
* **建議做法：** 護照技能將調整為符合素養先於技術的內容，例如：  
  * 【公民意識】能說出「數位足跡」的意義  
  * 【身份認證】成功登入 smail  
  * 【打字萌芽】中打一分鐘能正確輸入 20 個字  
  * 【資訊探尋】能獨立完成「圖書館尋寶」的手寫紀錄單

### **【八年級：專題應用與資訊思辨深化階段】**

* **目標：** 銜接七年級的基礎技能，在八年級的專題課程中，進一步深化資訊的應用、搜尋、整理與思辨能力，並**強調「歷程」的重要性**。  
* **執行模式：** 在八年級專題課程中，當有資訊處理需求時，由資訊老師入班進行補充教學。  
* **建議內容：**  
  1. **網路案例討論：** 介紹網路活動的實際案例（如 AI 倫理、假訊息傳播等），進行討論。  
  2. **進階資料搜尋 (導入歷程限制)：** 當專題需要蒐集資料時，資訊老師除了補充搜尋技巧外，必須**導入以下歷程限制**，確保學生專注於「探尋」而非「複製」：  
     * **限制時間：** 在操作資訊工具進行搜尋時，**有意識地限制時間**。  
     * **轉化為紙本：** 學生必須將查找到的資料，**列印出紙本資料**。  
     * **紙本索引：** 學習如何**整理並索引紙本資料**，並應用於專題製作，避免「快速廉價的產出報告」。  
  3. **網路訊息辨識與媒體素養：** 將目前在九年級資訊課探討的內容，濃縮精華，於導師課的時間帶領學生思辨。

### **三、 行政與設備配套建議**

1. **教師共備與支持：** 此方案的成功，需要資訊老師與導師、科任老師的緊密合作。建議由資訊老師在學期初召集共備會議，說明理念與操作方式，並提供「任務範例包」與支援管道。  
2. **設備與管理：**  
   * **硬體配置：** 建議可在**各班級設置公用電腦**，以利學生隨時練習與完成任務。  
   * **帳號管理：** 公用電腦可**區分教師與學生登入帳號**，便於管理。  
   * **降低導師負擔：** 為減少導師的管理負擔，公用電腦可設定「**白名單**」，限制學生僅能連結至指定的學習網站（如打字練習、圖書館網站、宜蘭快樂e學院等），避免不當使用。  
3. **家長溝通：** 在學期初的家長日，由導師向家長說明此「融入式學習方案」的理念與長程規劃，特別是「素養優先」與「強化線下歷程」的用心，爭取家長認同與配合。
